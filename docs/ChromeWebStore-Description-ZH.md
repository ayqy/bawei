# bawei - 微信公众号文章一文多发（多平台同步发布）

bawei 是一款 Chrome 浏览器插件：当你打开微信公众号文章页（`mp.weixin.qq.com`）时，它会在页面内注入发布面板，一键抓取文章标题与正文，并同步到多个平台进行“保存草稿”或“直接发布”。

## 核心能力

- 一键提取微信公众号文章：标题、正文（渲染后的 HTML）、原文链接（当前 URL）
- 多平台并发：同时打开所选平台编辑页并自动填充标题与正文
- 原创优先：当平台不提供“原文链接/来源”字段或原创模式不可用时，会将 `原文链接：<url>` 追加到正文末尾
- 诊断与提示：遇到未登录/验证码/实名/必填项/风控等阻塞时，在面板中给出提示与建议，用户处理后可继续/重试

## 支持的平台（V2）

- CSDN
- 腾讯云开发者社区
- 博客园（CNBlogs）
- OSCHINA
- 人人都是产品经理（WoShiPM）
- 墨问
- 少数派（SSPAI）
- 百家号
- 头条号
- 飞书文档

## 使用方式

1. 打开任意微信公众号文章页（`mp.weixin.qq.com`）
2. 在页面中打开发布面板，选择动作（保存草稿/直接发布）与渠道
3. 点击开始：插件会并发打开各平台编辑页并执行填充
4. 若出现平台限制提示：按面板建议完成登录/验证/补全必填项后，点击“继续/重试”

## 隐私与权限说明

- 纯本地运行：不需要服务器，不上传你的文章内容
- 需要的权限仅用于：读取当前微信文章内容、打开目标平台标签页、在目标页面执行自动化填充与验收

## 说明

- 图片策略（当前版本）：不上传图片，默认复用原文外链；若平台限制外链导致图片不可用，请按面板提示手动处理。
